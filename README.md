# OME-D3R
OME-D3R是一个基于开源Bio-Formats工程的成像分析工具，由Matlab语言编写，主要是用于常规细胞轮廓光滑重建，以及成像靶标相关的定量化分析。通过模块集成的、可设计的、批处理的设计，OME-D3R（如图1）可以方便地针对相关统计特征，自动化进行分析、采集、作图和转存为Matlab无缝共享的数据以便再分析的工作。这样一来，用户可以集中精力在所关注的生物问题的关联分析工作中，而不必困扰于繁重的鼠标操作。再者，高级用户也可基于现有框架，进行再开发工作，定制自己的分析模块。
